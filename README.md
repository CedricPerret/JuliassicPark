# 🦖 JuliassicPark.jl

**JuliassicPark.jl** is a lightweight and flexible Julia package for simulating agent-based evolutionary models with customisable fitness functions. It’s a toolbox for building your own digital ecosystems — evolution, selection, and adaptation all in your hands. 

---

## 🚧 Status

This package is **not yet ready for public use**. It is currently under development for **personal use** only. 

---

## 📦 Structure

- `utility.jl`: Core reusable functions (e.g., utility functions, exchange rules, etc.)
- `toolbox.jl`: Components and logic for running evolutionary simulations
